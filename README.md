# AI客户智能评分系统

一个基于Web的智能客户评分与排序工具，专为AI提效咨询公司设计，帮助销售团队识别高价值客户并优化跟进策略。

## 功能特点

### 🎯 核心功能
- **客户信息管理**：添加、编辑、删除客户记录
- **AI智能评分**：基于多维度因素自动计算客户价值
- **优先级排序**：根据AI评分智能排序客户
- **跟进建议**：生成个性化的客户跟进策略
- **数据分析**：实时统计与可视化展示

### 📊 评分维度
1. **潜在价值** (40%权重) - 客户长期合作潜力
2. **年营业额** (30%权重) - 客户经济实力
3. **互动程度** (20%权重) - 现有关系密切程度
4. **紧急程度** (10%权重) - 需求紧迫性
5. **行业与规模加成** - 行业前景和公司规模调整系数

### 🚀 技术特性
- **本地存储**：数据保存在浏览器本地，无需服务器
- **响应式设计**：适配桌面和移动设备
- **实时分析**：添加客户后立即更新统计数据
- **数据导出**：支持分析报告导出为文本文件
- **批量导入**：支持Excel/CSV文件批量上传
- **示例数据**：内置示例客户数据，快速体验

## 快速开始

### 方法一：直接打开（简单测试）
1. 下载项目文件到本地
2. 用浏览器打开 `index.html`
3. 开始添加客户数据

### 方法二：本地服务器（推荐）
**Windows用户**：
1. 双击 `start.bat` 文件
2. 等待依赖安装和服务器启动
3. 浏览器访问 `http://localhost:3000`

**Mac/Linux用户**：
```bash
# 给脚本执行权限
chmod +x start.sh

# 运行启动脚本
./start.sh

# 或手动安装和启动
npm install
npm start
```

### 方法三：开发模式
如需修改或扩展功能：
1. 确保有代码编辑器（如VS Code）
2. 确保已安装 Node.js
3. 打开项目文件夹，运行开发服务器：
```bash
npm install
npm run dev
```
4. 访问 `http://localhost:3000`

### 方法四：使用静态文件服务
```bash
npm install -g serve
serve -s . -p 3000
```

## 使用指南

### 1. 添加客户
#### 方式一：手动添加
填写表单中的客户信息，包括：
- 客户姓名/公司名
- 行业类型
- 公司规模
- 年营业额（万元）
- 各项评分（1-10分）
- 备注信息

#### 方式二：批量导入（推荐）
1. 下载模板文件 `sample_customers.csv`
2. 在Excel中按模板格式填写客户数据
3. 将文件拖放到上传区域或点击"选择文件"
4. 系统自动解析并导入所有客户
5. 支持格式：.xlsx, .xls, .csv

### 2. AI分析
点击"开始AI分析"按钮，系统将：
- 计算每个客户的综合评分
- 确定优先级（高/中/低）
- 生成跟进建议
- 自动排序客户列表

### 3. 查看结果
- **统计卡片**：查看总体数据概览
- **AI推荐**：获取系统生成的洞察建议
- **排序表格**：按不同维度排序客户
- **优先级标签**：快速识别高价值客户

### 4. 导出报告
点击"导出报告"按钮，生成包含：
- 客户列表及详细信息
- AI评分和优先级
- 统计摘要
- 跟进建议

## 技术实现

### 评分算法
```javascript
总分 = (潜在价值×4 + 营业额评分×3 + 互动程度×2 + 紧急程度×1) × 行业系数 × 规模系数
```

### 数据存储
- **本地存储**：使用浏览器localStorage保存数据
- **数据格式**：JSON格式，包含完整客户信息
- **持久化**：关闭浏览器后数据不会丢失

### 优先级划分
- **高优先级**：AI评分 ≥ 80分
- **中优先级**：AI评分 ≥ 60分
- **低优先级**：AI评分 < 60分

## 自定义配置

### 修改评分权重
在 `script.js` 文件中找到 `analyzeCustomers` 函数，修改权重参数：
```javascript
// 当前权重设置
score += customer.potentialValue * 4;      // 40%
score += revenueScore * 3;                 // 30%
score += customer.engagementLevel * 2;     // 20%
score += customer.urgency * 1;             // 10%
```

### 调整行业系数
在 `industryBonus` 对象中修改各行业的加成系数。

### 添加新的评分维度
1. 在HTML表单中添加新字段
2. 在JavaScript中读取和处理新字段
3. 更新评分算法逻辑

## 常见问题

### Q: 数据安全吗？
A: 所有数据仅保存在本地浏览器中，不会上传到任何服务器。

### Q: 支持多少客户记录？
A: 理论上受浏览器localStorage限制（通常5-10MB），足够存储数千条客户记录。

### Q: 能导出Excel格式吗？
A: 当前版本支持导出为文本格式，可扩展为Excel导出功能。

### Q: 如何批量导入客户数据？
A: 系统支持Excel/CSV文件批量导入：
1. 下载模板文件了解数据格式
2. 准备客户数据文件
3. 通过拖放或点击选择文件上传
4. 系统自动解析并导入所有客户

### Q: 如何重置数据？
A: 在浏览器控制台执行 `AICustomerSystem.clearAllData()` 或清除浏览器本地存储。

### Q: API密钥无效怎么办？
A: 如果遇到401未授权错误，请：
1. 运行检查脚本：`node check-api-key.js`
2. 或切换到模拟模式：运行 `switch-to-mock-mode.bat`
3. 详细解决方案请参考 [API_KEY_SOLUTION.md](API_KEY_SOLUTION.md)

### Q: 如何获取有效的Deepseek API密钥？
A: 
1. 访问 https://platform.deepseek.com/
2. 注册账号并登录
3. 进入 "API Keys" 页面创建密钥
4. 更新 `.env` 文件中的 `DEEPSEEK_API_KEY`
